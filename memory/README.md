# memory 学习资料目录

这个目录用于保存当前播放器学习项目的长期记忆和复习资料。

## 文档列表

### 1. 项目学习记忆文档

文件：`project-study-notes.md`

内容包括：

- 项目定位
- 技术栈
- 文件架构
- 后端每个文件的职责
- Next.js 前端每个文件的职责
- 上传、列表、播放、删除的数据流
- 当前项目重点练习的知识点
- 常用启动和检查命令
- 后续可以继续练习的方向

### 2. Bug 复盘：播放页返回首页后视频叠音

文件：`bug-video-audio-overlap.md`

内容包括：

- Bug 现象
- 排查过程
- 根本原因
- 最终修复方案
- React / Next.js 生命周期相关知识点
- HTML video 媒体资源释放经验

### 3. 边界校验学习记录：上传视频表单

文件：`validation-notes.md`

内容包括：

- 前端校验和后端校验的区别
- 标题、介绍、视频、封面的边界规则
- `null` / `undefined` / `NaN` 字符串处理
- 文件大小、MIME 类型、扩展名校验
- multer 上传限制

### 4. 数据库与存储桶同步学习记录

文件：`storage-sync-notes.md`

内容包括：

- 数据库和存储桶为什么不会自动同步

- 删除接口为什么要注意顺序
- 如何用脚本清理孤儿文件
- S3 / MinIO 列出对象和删除对象的基础思路

## 建议阅读顺序

如果你是复习整个项目，建议按这个顺序看：

```txt
1. 项目定位
2. 技术栈
3. 文件架构说明
4. 数据流说明
5. 当前项目重点知识点清单

```
